How to Create Your Own Web Page in GitHub

You can start creating your own webpage in GitHub by follow some easy steps below:

1. Go to GitHub website to create an account

2. Create a new repository

3. Add Files and Commit Changes

4. Make sure the file is added and launch webpage

Now that you have done adding your "index.html" file to your GitHub repository, you can simply just go to your browser and put "yourusername.github.io" you will be able to see your webpage.
